# Chapter 7: Project Proposal Pitch

## Summary

This chapter brings together everything from the planning phase into a formal engineering proposal. Students learn how to structure and write a compelling proposal, develop the presentation and visual communication skills needed to defend it, and then experience the peer review process from both sides — as presenters and as critical reviewers providing constructive feedback.

## Assignment

**Project Proposal Pitch Presentation** — CTE-52, CTE-53

## Concepts Covered

This chapter covers the following 8 concepts from the learning graph:

1. Presentation Skills
2. Visual Communication
3. Proposal Writing
4. Project Proposal Structure
5. Proposal Review Criteria
6. Constructive Feedback
7. Proposal Defense
8. Peer Review Process
